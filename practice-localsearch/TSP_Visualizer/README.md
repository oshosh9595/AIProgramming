## Local Search 알고리즘
- 중간결과값을 계속받아야한다
- 중간과정을 모니터링(관찰) 할수있다. => 중간결과를 표현 할 수 있음
    - 식사하는 철학자 문제 => Queue => 공유자원 할당 문제
    - 생산자/소비자 문제 => DeQueue => 공항 활주로 착륙
